## Installation

1. Clone the repo
```sh
git clone https://github.com/oluwatunmiisheii/wlabel.git
```
2. Open the project Directory in your local machine
```
cd health-app
```
1. Install NPM packages
```sh
npm install
```
4. Run Application
```JS
npm start
```
5. Run Tests
```JS
npm test
```
